=== Описание структуры папки memory ===

Папка memory/ предназначена для сохранения контекста проекта.

Файлы в папке:

- PROJECT_START_PROMPT.txt — общее описание проекта и шаги для старта нового ИИ или человека.
- DEV_PLAN.txt — дорожная карта проекта, история изменений.
- COMMUNICATION_PROTOCOL.txt — правила общения, инструкции для взаимодействия с ИИ.
- HUMAN_EVOLUTION_LOG.txt — наблюдения за развитием навыков участника проекта.

=== Как развернуть memory в другом проекте ===

1. Создать папку memory/ в корне проекта.
2. Поместить в неё копии всех файлов выше.
3. Настроить ChatGPT (или другого ИИ) изучить файлы из memory перед началом работы.
4. Всегда сопровождать изменения обновлением этих файлов.

=== Основная философия ===

Проект — это не только код. Это ещё и:
- Путь развития.
- Стиль общения.
- История решений.

Папка memory обеспечивает сохранение всех этих аспектов.

# Обязательные протоколы:

- Каждый ИИ обязан ознакомиться и следовать `INSPECTION_PROTOCOL.md`.
- Нарушение инспекционного регламента (например, просьбы вставить содержимое файла вручную при наличии ссылки) считается критической ошибкой и ведёт к перезапуску чата.
