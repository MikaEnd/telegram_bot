=== Протокол общения с ChatGPT ===

1. Все инструкции начинаются с авторизации по SSH:
   ssh -l avipython 89.169.160.147

2. Переходить в проект командой:
   cd ~/telegram_bot

3. Все команды, редактирования и запуск скриптов подаются целиком, без фрагментов.

4. Все файлы редактируются полностью (целиком), с помощью nano или другим редактором.

5. После любых изменений выполняется:
   ./sync_github.sh → 2 (Push на GitHub)

6. ChatGPT при обнаружении запуска sync_github.sh считает это сигналом, что проект обновлён, и обязан свериться с GitHub.

7. Если sync_github.sh отсутствует — создаётся символическая ссылка:
   ln -s ~/sync_github.sh ~/telegram_bot/sync_github.sh

8. Команды и действия фиксируются в соответствующих файлах памяти:
   - DEV_PLAN.txt — история разработки
   - HUMAN_EVOLUTION_LOG.txt — поведение и стиль
   - PROJECT_START_PROMPT.txt — структура и миссия проекта

9. После любого важного шага ChatGPT автоматически:
   - предлагает обновление протоколов
   - даёт команды по фиксации
   - сопровождает push в GitHub
