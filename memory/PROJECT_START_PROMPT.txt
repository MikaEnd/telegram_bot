=== Описание проекта ===

Проект: Telegram-бот-помощник для управления сервером (VM на Yandex.Cloud).
Язык: Python 3
Технологии: Telegram API, Paramiko SSH, systemd сервисы, GitHub синхронизация.

Цель: Создать интеллектуального ассистента для контроля сервера и развития навыков ИИ.

=== Основные файлы памяти ===

- PROJECT_START_PROMPT.txt — описание проекта и протоколов.
- DEV_PLAN.txt — история этапов разработки.
- COMMUNICATION_PROTOCOL.txt — строгое описание взаимодействия с проектом.
- HUMAN_EVOLUTION_LOG.txt — лог развития навыков пользователя.
- MEMORY_README.txt — описание структуры памяти.

=== Порядок действий для нового ИИ или разработчика ===

1. Авторизоваться через SSH командой:
   ssh -l avipython 89.169.160.147

2. Перейти в папку проекта:
   cd ~/telegram_bot

3. Все действия выполнять строго по полным инструкциям, начиная с описания полного пути команд.

4. Все изменения файлов выполняются полным текстом через редактор (например, nano).

5. После любых изменений:
   - Обновить DEV_PLAN.txt (описать этап).
   - Обновить HUMAN_EVOLUTION_LOG.txt (описать ошибки и успехи).
   - Сделать push через ./sync_github.sh

6. Соблюдать правила из COMMUNICATION_PROTOCOL.txt без интерпретаций.

=== Особенности ===

- Запрещено обобщать протоколы: только механическое следование.
- Запрещено выполнять команды частично или с сокращениями путей.
- После каждого значимого действия обязательно фиксировать его в памяти.
