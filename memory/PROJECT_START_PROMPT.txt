=== Описание проекта ===

Проект: Telegram-бот-помощник для управления сервером (VM на Yandex.Cloud).
Язык: Python 3
Технологии: Telegram API, Paramiko SSH, systemd сервисы, GitHub синхронизация.

Цель: Создать интеллектуального ассистента для контроля сервера и развития навыков ИИ.

=== Основные файлы для понимания проекта ===

- telegram_bot/  — Основная папка с кодом
- memory/DEV_PLAN.txt — История всех этапов развития
- memory/COMMUNICATION_PROTOCOL.txt — Протокол общения
- memory/HUMAN_EVOLUTION_LOG.txt — Стиль работы пользователя

=== Где искать код ===

Репозиторий: https://github.com/MikaEnd/telegram_bot

=== Особенности взаимодействия ===

- Все изменения идут через GitHub push.
- Все инструкции идут полными файлами.
- Контекст обновляется через память в папке memory/.

=== Что делать первому ИИ или человеку ===

1. Изучить PROJECT_START_PROMPT.txt полностью.
2. Перейти к изучению DEV_PLAN.txt и COMMUNICATION_PROTOCOL.txt.
3. Затем открыть telegram_bot/ и смотреть актуальный код.
